## 🎯 증강 데이터 활용 전략

### 단계별 활용
1. **프롬프트 초안 작성** → 원본 46개로 테스트
2. **기본 증강 적용** → 200개로 확장 테스트  
3. **경계 케이스 추가** → 어려운 케이스들로 성능 향상
4. **트릭 케이스 투입** → 최종 정확도 극대화

### 품질 관리 체크리스트
- [ ] 문법적 정확성 확인
- [ ] 의미 변화 없음 확인  
- [ ] 라벨 적정성 재검토
- [ ] 중복/유사도 검사
- [ ] 현실성/그럴듯함 검증

## 💡 핵심 인사이트

1. **경계 케이스가 핵심**: 배터리/반도체의 용도별 분류
2. **트릭 케이스로 강화**: 함정을 피하는 판단력 훈련  
3. **다양성 확보 필수**: 동일 패턴 반복 방지
4. **현실성 유지**: 말이 되는 시나리오만 생성

이렇게 체계적으로 증강된 데이터로 프롬프트를 훈련시키면 훨씬 정교하고 안정적인 분류 성능을 얻을 수 있습니다!
