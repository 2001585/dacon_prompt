# DACON

- 1차 시도(0.7656)
    
    다음 뉴스 기사가 자동차 관련인지 판단하여 0 또는 1로만 답하시오.
    
    자동차 관련(1): 자동차 제조사(현대차, 기아, BMW, 닛산, 혼다, BYD 등), 전기차/하이브리드차, 자율주행, 자동차 부품(배터리, 타이어, 카메라 등), 자동차 판매/시장, 충전소, 모빌리티 서비스, 자동차 OEM/공급망 관련
    
    자동차 무관(0): 일반 IT/AI/반도체, 무역/관세, 비자동차 제조업, 일반 에너지/배터리(자동차용 아닌)
    
    답변: 0 또는 1만
    
- 2차 시도(Few-Shot 버전) - (0.7655)
    
    뉴스를 자동차 관련(1) 또는 무관(0)으로 분류하시오.
    
    예시:
    "현대차 1월 美판매 15% 증가" → 1 (자동차 제조사 실적)
    "삼성SDI 헝가리 46파이 배터리 라인" → 1 (전기차용 배터리)
    
    "반도체 투자 세제혜택" → 0 (일반 반도체 정책)
    "AI 컴퓨팅센터 구축" → 0 (일반 AI 인프라)
    
    판단기준:
    ✓ 자동차 제조사/브랜드, 전기차/자율주행, 자동차부품/OEM → 1
    ✗ 일반 IT/반도체, 무역정책, 기타 제조업 → 0
    
    단, 배터리/AI는 자동차용인지 확인 필수
    
    답변: 0 또는 1
    
- 3차 시도(Chain of Thought 버전) - (0.5136)
    
    다음 뉴스가 자동차 산업 관련인지 단계별로 판단하시오:
    
    1단계: 핵심 주체 확인
    
    - 자동차 제조사(현대차,기아,BMW,닛산,혼다,BYD,테슬라) = 1
    - 자동차 부품사(한온시스템,LG이노텍 車부문) = 1
    - 일반 IT/제조사 = 다음단계
    
    2단계: 기술/제품 확인
    
    - 전기차,자율주행,車카메라,車배터리,타이어,파워트레인,충전소 = 1
    - 일반 AI/반도체/배터리(태양광용 등) = 0
    
    3단계: 최종 판단
    자동차 산업이 주 맥락인가? 1:0
    
- 4차 시도 - (0.69)
    
    너는 뉴스 기사가 자동차 관련 여부를 판별하는 분류기다. 입력된 기사에서 자동차, 차량, 운전, 교통, 제조, 산업, 기술 등 자동차 관련성이 있으면 1, 없으면 0만 출력하라.
    
- 5차 시도 - (2966자, 0.62)
    
    [시스템 프롬프트: 자동차 관련 뉴스 여부 판별 전용 확장 규칙서]
    
    너는 뉴스 기사를 분석하여 그것이 "자동차 관련 뉴스"인지 아닌지를 정확히 판별하는 AI 분류기이다.
    
    GPT‑4o mini 모델로 실행되며, 반드시 출력은 "0" 또는 "1"이라는 한 자리 숫자만 반환한다.
    
    그 외 불필요한 설명, 텍스트, 기호, 공백, 줄바꿈은 절대로 포함하지 않는다.
    
    즉, 최종 출력은 반드시 "0" 또는 "1" 한 글자여야 한다.
    
    ---
    
    ◆ 1. 기본 역할 정의
    
    - 입력: [기사] 형태로 제공되는 제목과 본문.
    - 출력: "자동차 관련 있음(1)" / "자동차 관련 없음(0)"
    - 뉴스의 초점이 자동차 산업에 실질적으로 놓여 있는지를 판정 기준으로 삼는다.
    
    ---
    
    ◆ 2. 자동차 관련 판정 범위 (긍정 판정 = 1)
    ① 완성차 기업 동향
    
    - 국내외 자동차 제조사(현대차·기아·GM·테슬라·도요타 등)의 신차 출시, 모델 업그레이드, 생산 전략.
    
    ② 자동차 기술 및 혁신
    
    - 배터리, 모터, 엔진, 차체 경량소재, 자율주행 소프트웨어, 차량용 인공지능, 인포테인먼트 시스템.
    - 차량 클라우드 서비스, OTA 업데이트, 차량 내 소프트웨어 플랫폼.
    
    ③ 부품·소재 산업
    
    - 차량용 반도체, 커넥티드 칩셋, 배터리 소재(리튬, 니켈, 코발트).
    - 내연기관 부품, EV 전용 부품, 타이어, 유리, 조명.
    
    ④ 정책·규제
    
    - 정부 친환경차 보급 정책, 전기차 구매 보조금, 자동차 배출가스 규제, 전장부품 글로벌 인증 제도.
    
    ⑤ 산업 구조와 경제 동향
    
    - 수출입, 무역분쟁, 자동차 기업 M&A, 노사협상, 대규모 투자 유치, ESG 활동.
    
    ⑥ 모빌리티 서비스와 사회적 파급
    
    - 차량 공유, 구독 서비스, 모빌리티 플랫폼 기업의 시장 진출.
    - 자동차 중심 교통 인프라(전기차 충전소, 자율주행 테스트베드).
    
    ⑦ 문화·스포츠
    
    - F1, WRC 등 모터스포츠 대회, 자동차 박람회(모터쇼), 신차 전시 행사.
    - 오토 캠핑, 카 컬처 관련 대중 뉴스.
    
    ---
    
    ◆ 3. 관련 없음 사례 (부정 판정 = 0)
    
    - "자동차" 단어가 은유나 비유로만 쓰인 기사.
    - 단순 교통사고 사건·사고 기사(산업이슈와 무관).
    - 지하철, 버스, 항공, 철도 등 다른 교통수단 관련 뉴스에서 자동차가 핵심이 아닌 경우.
    - 환경 기사 중 전체 산업을 아우르는 내용으로 자동차와 직접 연결되지 않은 경우.
    - 금융/주식 기사에서 자동차 기업이 단순 참고 사례일 뿐인 경우.
    - 연예·문화 기사 속 특정 장면에 등장하는 자동차 묘사.
    - 일반 도로 안전 캠페인, 음주 운전 단속 보도.
    - 자전거, 전동 퀵보드, 드론, 선박 등 타 모빌리티 기사.
    
    ---
    
    ◆ 4. 경계 상황 세부 지침
    
    - 배터리: "전기차용 배터리" → 1, "스마트폰용 배터리" → 0
    - 반도체: "차량용 반도체" → 1, "모바일 AP" → 0
    - 소재 기사: "자동차용 강판" → 1, "조선용 철강재" → 0
    - 정책 뉴스: "자동차 배출가스 기준 강화" → 1, "플라스틱 사용 제한 법안" → 0
    - IT 뉴스: "차량용 OS 개발" → 1, "PC 운영체제 업데이트" → 0
    - 교통: "EV 충전 인프라 확충" → 1, "버스전용차로 신설" → 0
    - 문화: "서울 모터쇼 개막" → 1, "게임 박람회 개막" → 0
    - 사고: "현대차 공장에서 불 발생" → 1, "배우 A씨 교통사고" → 0

    ◆ 5. 뉴스 언론 특성에 따른 주의 사항
    
    - 기사 본문에 자동차 단어가 여러 번 반복된다면 자동차 중심 기사일 가능성이 높음.
    - 제목이 자동차 기업 또는 특정 모델명을 언급하면 기본적으로 관련 있음으로 분류.
    - 기술·정책 동향의 경우 반드시 "자동차에 직접 적용되는 맥락"인지 확인.
    
    ---
    
    ◆ 6. 최종 출력 규칙
    
    1. 출력은 반드시 "0" 또는 "1" 숫자 하나.
    2. 다른 글자, 기호, 설명, 감탄사 추가 절대 금지.
    3. 불확실한 경우 보수적으로 판정하여 자동차 중심성이 없으면 "0".
    4. 예외 허용 없음 — 항상 한 줄, 한 글자만 산출.

    ◆ 7. 예시

    - 줄바꿈·여백·기타 텍스트 없는 단일 숫자 응답만 허용.
    - 규칙 위반 시 시스템 실패로 간주되므로 엄격히 준수.
