# 김경태 프롬프트가 0.9801 나온 이유 분석

## 🎯 핵심 성공 요인

### 1. **게이트 조건** (가장 중요!)
```
total≥3 이면서 (① OEM/차종/차량용/규제·인증 신호 중 하나 명시 또는 ② A∧Act 동시문장) 일 때만 1
```
- 단순 점수 합계가 아닌 **이중 조건**
- 점수가 높아도 게이트 조건 없으면 0
- 이것이 정확도를 크게 높임

### 2. **음수 점수 시스템**
```
-3 제목 B 중심(자동차 연결 없음)
-2 본문 B 중심(직접성 불명)
-2 배터리·반도체·소재·에너지: '차량용' 불명
```
- 비자동차 산업을 **적극적으로 배제**
- 애매한 케이스를 0으로 밀어냄

### 3. **A∧Act 동시 출현 체크**
```
+1 A∧Act 동일 문장(근접)
```
- 주체와 행위가 **같은 문장**에 있어야 점수
- 문맥 파악의 핵심

### 4. **'차량용' 명시 중요도**
```
+1 차량용/자동차용/오토모티브/AEC-Q/ISO26262
-2 배터리·반도체·소재·에너지: '차량용' 불명
```
- 차량용 명시 시 +1
- 차량용 불명 시 -2
- **3점 차이로 판정 뒤집힘**

## 🔴 Llama-3.2에서 실패한 이유

### Sample 33 분석 (모두 실패)
```
제목: "뉴로틴, 반도체 스타트업이 만든 육아··오디오북 앱"
정답: 1 (자동차 관련)
모든 프롬프트 예측: 0
```

**왜 자동차 관련인가?**
- 본문에 자동차 관련 내용이 있을 가능성
- 제목만으로는 판단 불가
- Llama-3.2는 본문 이해력 부족

### M12 축약스코어링 실패 원인
```
[車분류]"1"또는"0"만
```
- 너무 짧아서 맥락 이해 못함
- 게이트 조건 설명 부족
- 수학 기호 이해 못함

## 💡 개선 방향

### 1. 게이트 조건 강화
```
[최종판정]
점수≥3 AND (차량용 명시 OR (완성차기업 AND 자동차행위))
→ 1
나머지 → 0
```

### 2. 명확한 우선순위
```
[우선순위]
1. 완성차기업(현대차/기아/테슬라) → 즉시 1
2. 차량용 명시 → 거의 확실히 1
3. ESS/UAM/항공 → 즉시 0
4. 애매하면 → 0
```

## 🚀 최종 제안

### V3 프롬프트 (500자)
```
[자동차뉴스분류] "1" 또는 "0"만

[집합정의]
A=완성차·OEM·전장·부품·타이어·충전·차량용배터리
Act=출시·양산·생산·투자·수주·공급계약·판매

[점수계산]

+2 행위가Act
+1 차량용명시
+1 A와Act동시출현
-3 B중심
-2 차량용불명확

[판정규칙]
합계≥3 그리고 (차량용명시 또는 A와Act동시)→1
그외→0
모호→0
```

- 김경태 원본의 핵심 유지
- 500자로 압축
