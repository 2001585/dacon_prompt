# 1단계: 데이터 분석

## 목표
제공된 예시 데이터를 분석하여 자동차 관련 뉴스의 특성과 패턴을 파악

## 작업 내용

### 1. 학습 데이터 분석
- [ ] 제공된 예시 데이터 확인
- [ ] 자동차 관련 뉴스(1)의 특징 분석
  - 키워드 패턴
  - 제목의 특성
  - 내용의 특성
- [ ] 자동차 무관한 뉴스(0)의 특징 분석
  - 오분류 위험이 높은 케이스 식별
  - 혼동하기 쉬운 주제들 정리

### 2. 패턴 도출
- [ ] 자동차 관련 핵심 키워드 리스트 작성
- [ ] 부정적 키워드(제외할 키워드) 리스트 작성
- [ ] 문맥상 판단이 필요한 애매한 케이스 정리

### 3. 분류 기준 설정
- [ ] 명확한 분류 규칙 정의
- [ ] 경계 케이스 판단 기준 수립
- [ ] 예외 상황 처리 방안 마련

## 산출물
- 데이터 분석 보고서
- 키워드 매트릭스
- 분류 기준 가이드라인
