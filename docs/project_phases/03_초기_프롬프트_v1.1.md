**출력 규칙:**
- 자동차 관련: 1
- 자동차 무관: 0  
- 반드시 숫자만 출력, 추가 설명 금지

산업: 자동차공장,자동차생산,판매대수,출고량,자동차시장,점유율,자동차수출

**Tier 2: 맥락 의존 키워드 (용도 확인 필요)**

기술: 배터리,반도체,AI,센서,카메라,디스플레이 → 자동차 용도?
회사: 삼성,LG,SK,한화,포스코 → 자동차 사업?  
일반: 생산,제조,투자,공장,R&D → 자동차 관련?

**Tier 3: 비자동차 신호 (거의 확실히 0)**

분야: 부동산,금융,은행,보험,증권,정치,외교,군사,국방,우주,항공,조선,의료,병원,교육,대학,게임,엔터테인먼트,문화,스포츠,요리,패션

기술: 스마트폰,컴퓨터,서버,클라우드,웹서비스,앱개발,블록체인,가상화폐

**판단 알고리즘:**
1. 제목에서 Tier1 키워드 검색 → 발견시 거의 확실히 1
2. Tier3 키워드 검색 → 발견시 거의 확실히 0
3. Tier2 키워드 맥락 분석:
   - "전기차 배터리" → 1
   - "태양광 배터리" → 0  
   - "차량용 반도체" → 1
   - "서버용 반도체" → 0
4. 애매한 경우 보수적 판단 → 0

**예시:**
"현대차 전기차 판매 증가" → 1 (Tier1: 현대차,전기차)
"삼성SDI 가전용 배터리" → 0 (가전용=비자동차)
"LG전자 차량용 디스플레이" → 1 (차량용=자동차)
"네이버 AI 기술 개발" → 0 (Tier3: 포털)

반드시 0 또는 1만 출력하시오.
