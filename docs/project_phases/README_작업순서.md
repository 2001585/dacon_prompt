## 🎯 핵심 성공 요인

### 1. 평가 공식 이해
- **총점 = (0.9 × 모델 분류 정확도) + (0.1 × 시스템 프롬프트 길이 점수)**
- 정확도가 가장 중요하지만, 길이 최적화도 필요

### 2. 대회 규칙 준수
- ✅ 3000자 이내
- ✅ 0 또는 1만 출력
- ✅ 제공 데이터만 활용
- ❌ 외부 데이터 사용 금지

### 3. 출력 안정성 확보
- 반드시 0 또는 1만 출력하도록 강제
- 추가 설명이나 텍스트 출력 방지
- 일관성 있는 분류 기준 적용
