# 3단계: 초기 프롬프트 작성

## 목표
분석 결과와 설계 전략을 바탕으로 첫 번째 시스템 프롬프트 작성

## 작업 내용

### 1. 베이스라인 프롬프트 작성
- [ ] 기본 구조에 따른 프롬프트 초안 작성
- [ ] 글자 수 체크 (3000자 이내)
- [ ] 필수 요소 포함 확인
  - 역할 정의
  - 분류 기준
  - 출력 형식 (0 또는 1)

### 2. 핵심 키워드 통합
- [ ] 1단계에서 도출한 키워드 반영
- [ ] 자동차 관련 도메인 용어 포함
- [ ] 제외 키워드 명시

### 3. 분류 로직 구현
- [ ] 명확한 YES/NO 판단 기준 제시
- [ ] 복합적 판단이 필요한 경우 처리 방법
- [ ] 우선순위 기반 분류 체계

### 4. 출력 안정성 확보
- [ ] 출력 형식 강제 (0 또는 1만 출력)
- [ ] 추가 설명이나 텍스트 출력 방지
- [ ] 오류 상황 처리 방안

## 작성 시 체크포인트
- [ ] 3000자 이내 준수
- [ ] 모든 필수 요소 포함
- [ ] 명확한 분류 기준
- [ ] 안정적인 출력 형식
