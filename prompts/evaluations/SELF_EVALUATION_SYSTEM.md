```

```

### 즉시 0 조건 (OR 조건)
```
주제가 다음 중 하나라도 해당:
- 정치,정부,국방,우주,의료,교육,게임,문화
- 통신,포털,유통,건설,조선,항공,부동산,금융,투자
```

### 위험케이스 → 무조건 0
```
- 복합주제 (정부+자동차 등)
- 간접관련 (철강,화학,부품의부품)  
- 배터리/반도체/AI (용도 불명확)
- UAM,항공,선박,우주
```
