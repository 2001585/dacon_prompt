- [x] 제공된 예시 데이터 확인 (46개 샘플)
- [x] 자동차 관련 뉴스(1)의 특징 분석
  - [x] 키워드 패턴: 현대차, 기아, 삼성SDI, 전기차, SUV, 배터리 등
  - [x] 제목의 특성: "회사명 + 실적/기술" 패턴
  - [x] 내용의 특성: 구체적인 수치, 제품명, 시장 동향 포함
- [x] 자동차 무관한 뉴스(0)의 특징 분석

  - [x] 혼동하기 쉬운 주제들 정리: 제조업 일반, 무역/투자 일반

### 2. 패턴 도출
- [x] 자동차 관련 핵심 키워드 리스트 작성
  - 강한 신호: 현대차, 기아, 전기차, 자율주행, 완성차, OEM
  - 약한 신호: 배터리, 반도체, AI, 로봇 (맥락 의존)
- [x] 부정적 키워드(제외할 키워드) 리스트 작성
  - 비자동차: 부동산, 금융, 의료, 게임, 정치, 식품
- [x] 문맥상 판단이 필요한 애매한 케이스 정리
  - 배터리: 전기차용 vs 일반용
  - 반도체: 차량용 vs 일반용
  - 제조: 자동차 vs 일반 제조업

### 3. 분류 기준 설정
- [x] 명확한 분류 규칙 정의
  1. 직접적 연관: 자동차 회사/제품 명시
  2. 산업 특화: 자동차 특화 기술/시장
  3. 일반 산업: 자동차 언급 없는 일반 주제
- [x] 경계 케이스 판단 기준 수립
  - 우선순위: 제목 키워드 → 본문 구체적 언급 → 산업 맥락
- [x] 예외 상황 처리 방안 마련
  - 애매한 경우 보수적 접근 (맥락 부족시 0으로 분류)

### 2. 키워드 매트릭스
| 카테고리 | 강한 신호 (거의 확실) | 약한 신호 (맥락 의존) | 비관련 신호 |
|---------|---------------------|-------------------|------------|
| 회사명 | 현대차, 기아, 삼성SDI | LG, 삼성 (일반) | 네이버, 카카오 |
| 제품 | 전기차, SUV, 하이브리드 | 배터리, 반도체 | 스마트폰, PC |
| 기술 | 자율주행, 완성차, OEM | AI, 로봇, 센서 | 게임, 웹서비스 |
| 비즈니스 | 판매대수, 자동차공장 | 생산, 제조, 투자 | 부동산, 금융 |

### 3. 분류 기준 가이드라인

#### 자동차 관련(1) 판단 기준
1. **1차 확인**: 제목에 자동차 회사/브랜드 명시
2. **2차 확인**: 본문에 자동차 제품/기술 구체적 언급
3. **3차 확인**: 자동차 산업 특화 용어/맥락 존재

#### 자동차 무관(0) 판단 기준
1. **명확한 비관련**: 부동산, 의료, 게임, 정치 등
2. **일반 산업**: 자동차 특정 언급 없는 제조/IT
3. **애매한 경우**: 맥락 부족시 보수적으로 0

## 💡 주요 인사이트
1. **키워드 중심 분류의 한계**: 배터리, 반도체 등은 맥락이 중요
2. **제목의 중요성**: 대부분 제목에서 주제를 파악 가능
3. **보수적 접근 필요**: 애매한 경우 False Positive보다 False Negative가 안전
4. **회사명의 강력함**: 자동차 회사 언급시 거의 확실히 관련
5. **계층적 판단**: 제목 → 본문 → 맥락 순서로 판단
